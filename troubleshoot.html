📱 Instrucciones Adicionales

Si el problema persiste:

  1. Cierra todas las pestañas de la aplicación
  2. Cierra completamente el navegador
  3. Abre el navegador de nuevo
  4. Prueba abrir la app en modo incógnito
  5. Si funciona en incógnito, usa el "Reset Nuclear" arriba

Atajos de teclado útiles:

  • Ctrl + Shift + R (Windows) o Cmd + Shift + R (Mac) - Recarga sin caché
  • Ctrl + Shift + N (Windows) o Cmd + Shift + N (Mac) - Modo incógnito
  • F12 - Abrir consola de desarrollador
